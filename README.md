# Influgal

Influgal is a creator marketplace for discovering influencers, launching campaigns, managing applications, booking creators, and collaborating through campaign-aware messaging.

## Local development

Requirements:
- Node.js
- npm

Run locally:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```

## Main stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- Capacitor
